# SIGMA-Persebaran-Masjid-di-Bandar-Lampung
WebGIS Persebaran Masjid di Kota Bandar Lampung merupakan sistem informasi geografis berbasis web yang dirancang untuk menyajikan informasi mengenai lokasi masjid secara interaktif dan mudah diakses. WebGIS ini memanfaatkan teknologi Leaflet sebagai library pemetaan dan GeoJSON sebagai format penyimpanan data spasial.
